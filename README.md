# DevOps Nexus Portal

A comprehensive DevOps learning and management platform built with modern web technologies.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or bun package manager

### Installation

1. Clone the repository:
```sh
git clone <YOUR_GIT_URL>
cd devops-nexus-portal-main
```

2. Install dependencies:
```sh
npm install
# or
bun install
```

3. Start the development server:
```sh
npm run dev
# or  
bun run dev
```

The application will be available at `http://localhost:5173`

## Technologies Used

This project is built with:

- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library for building user interfaces
- **shadcn/ui** - UI component library
- **Tailwind CSS** - Utility-first CSS framework

## Features

- Modern DevOps tools and pipeline management
- Interactive learning curriculum
- Environment configuration management  
- Infrastructure monitoring dashboard
- Registration and scheduling system

## Development

- Run tests: `npm run test`
- Build for production: `npm run build`
- Preview production build: `npm run preview`
